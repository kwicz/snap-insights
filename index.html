Your privacy and data security are our top priorities. SnapInsights is designed with privacy-first principles.

‚úì

‚úì

No Third-Party Services

Extension operates independently without external APIs, analytics, or advertising networks.

Data Handling

  • Screenshots: Captured using Chrome's native captureVisibleTab API and saved directly to your Downloads folder
  • Annotations: Text and voice transcriptions are embedded as metadata in the image files
  • Settings: User preferences (mode selection, marker style) are stored locally using Chrome's storage API
  • Website URLs: Only captured as metadata for research context - not transmitted or stored externally

Commitment to Privacy

SnapInsights will never sell, share, or monetize your data. This extension exists solely to improve UX research workflows while respecting your privacy and maintaining complete data ownership.
